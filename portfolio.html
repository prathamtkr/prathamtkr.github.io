Helmet and Number plate detection system for on road safty

Project Supervisor: Dr. Gopal Singh Tandel

Aim of project was to design a pipeline to detect if a motorcycle driver is wearing helmet or not, if not then try to detect number plate from present multiple camera at a check point and extract the number from number plate using OCR (optical character recognition)

Challenge: Match the person not wearing helmet accuractely with correct number plate, this was acheived to a satisfactory accuracy using a algorithm to calculate the angle of bike respective to camera and so that number plate could be captured from different camera angles
